🧮 Calculadora de Idade!

Bem-vindo(a) à Calculadora de Idade, um aplicativo simples e eficiente criado com o objetivo de facilitar o cálculo da sua idade exata, com base na data de nascimento fornecida.

🎯 Objetivo da Aplicação

Nosso objetivo é oferecer uma ferramenta prática para calcular a idade do usuário de forma detalhada, exibindo o resultado em anos, meses e dias. Além disso, o sistema permite armazenar os cálculos realizados, garantindo praticidade para consultas futuras.

⚙️ Funcionalidades

    • 📅 Campo para inserir a data de nascimento no formato dd/mm/aaaa
      🔘 Botão “Calcular” para obter a idade exata
      🧾 Resultado exibido de forma clara: XX anos, YY meses e ZZ dias
      💾 Os dados são salvos no localStorage do navegador
    • 📊 Uma tabela exibe o histórico dos cálculos com opções para:
      🔍 Visualizar detalhes
      ✏️ Atualizar a data de nascimento
      🗑️ Excluir o registro

    • ✅ Validações

    A data de nascimento é obrigatória O sistema não permite datas futuras, garantindo a precisão dos resultados Apenas datas válidas são aceitas
